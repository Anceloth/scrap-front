## 🚦 **Inicio Rápido**

### 📋 **Prerrequisitos**
- Node.js `>= 16.0.0`
- npm o yarn

### ⚡ **Instalación en 30 segundos**

```bash
# 1️⃣ Clonar
git clone <repository-url>
cd scrap-front

# 2️⃣ Instalar
npm install

# 3️⃣ Configurar (opcional)
cp .env.local.example .env.local

# 4️⃣ ¡Ejecutar!
npm run dev
```
