## 🚦 **Quick Start**

### 📋 **Prerequisites**
- Node.js `>= 16.0.0`
- npm or yarn

### ⚡ **Installation in 30 seconds**

```bash
# 1️⃣ Clone
git clone <repository-url>
cd scrap-front

# 2️⃣ Install
npm install

# 3️⃣ Configure environment (optional)
cp .env.local.example .env.local
# Edit .env.local with your personal settings

# 4️⃣ Run!
npm run dev
```
